Tags HTML

(voltar)

Foram ensinadas algumas tags essenciais para uso no HTML, como:

Head, img src, h1, h2, p, blockquote, enfim para inicio de uso e interação com a linguagem de inicio são pertinentes para começo.

Os servidores leem o código HTML para entender e renderizar o conteúdo corretamente. Essa leitura é feita de cima para baixo, da mesma forma que você está lendo este artigo agora.

Você pode usar quantas tags quiser para formatar o conteúdo. No entanto, existem algumas regras essenciais sobre tags HTML que você precisará seguir.

Uma tag HTML deve conter três partes: Uma tag de abertura — começando com um símbolo < >; Conteúdo — instruções curtas para exibir o elemento na página; Uma tag de fechamento — terminando com um símbolo . Algumas tags HTML podem ser abertas. Isso significa que elas não precisam ser fechadas com . Essas tags são normalmente usadas para metadados ou quebras de linha.

Conclusão

(voltar)

Concluo que foi de suma importância este inicio com exemplos e conceitos bem didáticos, e que tendem a somar no conhecimento apresentado.

A primeira etapa do projeto consistia em pesquisa e compreendimento dos princípios mais elementares da linguagem HTML (Hypertext Markup Language). Essa é a linguagem padrão utilizada na criação e manutenção em sites na Web.

Através dessa linguagem podemos acrescentar dados, links, imagens, formulários, entre outros.

Após análise de sites do ramo tecnológico, foi possível adquirir referenciais para a elaboração de páginas deste segmento. Tanto em estruturação de página, quanto em seu layout.

A etapa seguinte foi fundamental para a definição de parâmetros e de como aplicar os recursos obtidos na primeira etapa.

A estruturação dos grandes sites de informática serviu de base para a aplicação de uma interface adequada ao atual mercado. Pois, apesar de sempre buscar inovações, elas seguem um padrão muito usado comercialmente. Isso se faz necessário para que seja mantida uma linguagem e estruturação de dinâmicas e de fácil interação.
